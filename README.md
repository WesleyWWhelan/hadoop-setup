# HADOOP Installation and Setup

### First - Install Hombrew:
https://brew.sh/

## Open Terminal Window:

### Second - Install Java:

`$ brew install java`

#### Check version: 

`$ java -version` 
( Should be 1.8)

#### Install JDK

`$ brew install adoptopenjdk8`

### Third - Install Hadoop:

`$ brew install hadoop`
